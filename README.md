# MyProgrammingPortfolio

This repository contains my C++ programs for the “Emerging Trends in Programming” assignment, including:

- Calculator Program
- ATM Program
- Number Guessing Game
- Grade Calculator

## How to Run Each Program

### Compile
g++ filename.cpp -o output

### Run
./output  (Linux/Mac)
.\output   (Windows PowerShell)
